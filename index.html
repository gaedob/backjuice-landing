Preguntas Frecuentes

¿BackJuice es realmente gratis?

Si, BackJuice Portal es 100% gratuito. No hay planes de pago, no hay publicidad, no hay limites de funcionalidades. Todas las caracteristicas estan disponibles para todos los usuarios.

¿Donde se guardan mis datos?

Los datos se guardan localmente en tu dispositivo. Tambien puedes sincronizar con Supabase (nube) para acceder desde multiples dispositivos y tener respaldo automatico. Tu controlas donde se almacena tu informacion.

¿Como funciona el envio de WhatsApp?

BackJuice genera un enlace de WhatsApp con el mensaje predefinido. Al hacer click, se abre WhatsApp Web o la app con el mensaje listo para enviar. Esto cumple con las politicas de WhatsApp y no requiere API adicional.

¿Puedo usar BackJuice para mi empresa/club/organizacion?

Si, aunque fue diseñado para cuotas escolares, BackJuice funciona perfectamente para clubes deportivos, juntas de vecinos, asociaciones y cualquier grupo que necesite administrar cuotas de sus miembros.

¿Que pasa si cambio de tesorero?

Si usas sincronizacion con Supabase, simplemente el nuevo tesorero inicia sesion con las credenciales del grupo. Si usas almacenamiento local, puedes exportar los datos y pasarlos al nuevo tesorero.
